# Book Scraper (Training Project)

This script scrapes book data from https://books.toscrape.com and saves it to a CSV file.



## Extracted Fields

* Title
* Category
* Price
* Availability
* Image URL



## Tech Stack

* Python
* requests
* BeautifulSoup
* pandas
